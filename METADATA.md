# WHAT IS INSIDE THE SUBSETDATA FOLDER?
The data in the SubsetData folder within this repository contains a subset of the Danish Gigaword Corpus. All files within the subset are plain text (txt) files. The Danish Gigaword Corpus (DAGW) consists of multiple sections and each section represents the source from which the data comes.
